## v0.1.0 — Initial release

**The Heath-Jarrow-Morton forward-rate framework from first principles** in
pure Python + NumPy/SciPy.  6 modules, ~700 lines, no rates library underneath.
The capstone of the rates pillar (tvm → fixedalt → shortrate → g2pp → HJM).

Built with the portfolio's **adversarial multi-agent workflow**: a 3-angle
design panel → synthesis into 18 algebraic identities → one refutation agent
per identity (**6 of 18 corrected** BEFORE implementation) → implementation →
adversarial code review.

### The 6 modules

  - **`curve.py`** — the initial curve P^M(0,T) (Flat / Quadratic /
    Nelson-Siegel) + the shared cancellation-free primitive B(z,τ)=-expm1/z.
  - **`vol.py`** — deterministic vol structures σ(t,T) with their integral
    S=∫σ: ConstantVol (Ho-Lee), ExponentialVol (Hull-White), MultiFactorVol.
  - **`drift.py`** — the HJM no-arbitrage drift α=ΣₖσₖSₖ, the bond-vol vector,
    and the deflated-bond martingale residual (sum of squares).
  - **`gaussian.py`** — curve-consistent bond price, option variance (B-product),
    the convexity self-block (Taylor-branched), and the Black-style ZCB option.
  - **`models.py`** — Ho-Lee / Hull-White / two-factor as vol structures + the
    cross-model reproductions recomputed inline (never importing a sibling).
  - **`simulate.py`** — Musiela forward-curve Monte Carlo under Q (exact
    transport via index shift), with martingale tests and a bias probe.

### Tests — 35/35 passing in ~46 s

All algebraic-identity based.  Headline identities:

  - the drift condition vs an independent quadrature; ∫α = ½Σₖ Sₖ² (NOT the
    square of the sum) makes the deflated bond a Q-martingale
  - curve consistency P_HJM(0,T)=P^M(0,T) exactly
  - **Hull-White ZCB-option vol == Vasicek option_sigma_p**, three ways (flagship)
  - **Ho-Lee bond carries r(t)**; **two-factor V == G2++ V(τ)**
  - put-call parity from independent forms; convexity Taylor branch
  - MC martingales E_Q[exp(−∫r)]=P^M(0,T), E^{Q^T}[P(T,S)]=P^M(0,S)/P^M(0,T)
